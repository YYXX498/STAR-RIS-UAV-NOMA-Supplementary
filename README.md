# STAR-RIS-UAV-NOMA-Supplementary
Supplementary materials for the paper on active STAR-RIS assisted UAV networks with NOMA
This repository provides supplementary materials for the paper:

**"On the Downlink Performance of Active STAR-RIS assisted UAV networks with NOMA"**

These files contain additional derivations, validations and benchmark discussions that support the revised manuscript.

## File List

- `S1_Weak_Direct_Link_Extension.pdf`  
  Extended investigation considering the weak direct link between the UAV and terrestrial users.

- `S2_Gamma_Approximation_Validation.pdf`  
  Validation of the moment-matching based Gamma approximation.

- `S3_Multi_User_Extension.pdf`  
  Discussion of a simplified multi-user NOMA extension.

- `S4_Benchmark_Comparisons.pdf`  
  Additional comparisons with benchmark schemes.

- `S5_Block_Varying_CSI_SIC.pdf`  
  Discussion of block-varying imperfect CSI and residual SIC.

- `S6_Multi_Antenna_UAV_Extension.pdf`  
  Extension to the multi-antenna UAV case.

## Remark

The main manuscript focuses on the fully blocked direct link case and the two-user paired NOMA scenario. The materials provided here are supplementary discussions used to further clarify the applicability and possible extensions of the proposed analytical framework.
